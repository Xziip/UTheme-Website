V0.3B

February 16, 2026

Lunar New Year Release

  • Multi-language Support: Now supports 13 different languages for a more global experience.
  • Search Functionality: Integrated swkbd support, allowing you to search for themes directly.
  • Random Theme: Added a "Random Theme" feature.
  • StyleMiiU Integration: Full support for StyleMiiU configuration. (Special thanks to @Fangal-Airbag!)
  • Storage Management: Added SD card remaining space detection for safe installations.
  • Safe Theme Saving: Themes are now saved using safe filenames.
  • Enhanced UI: Added a touch-screen back button.
  • Remote Notifications: Support for remote notification alerts.
  • JSON Parser Update: Replaced the simple parser with a more robust implementation.
  • Installation Optimization: Improved the theme installation logic to be faster and more reliable.
